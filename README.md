## Selected Work

### Casting Software  
AI powered casting and talent discovery platform  

A production system designed for image based search using vector embeddings. The platform enables similarity based discovery with a full pipeline from upload to ranking.

Architecture  

Multi stage image pipeline with preprocessing and background removal  
Embedding generation using CLIP based models  
Dual vector strategy for improved similarity accuracy  
Vector search using PostgreSQL with pgvector  
Hybrid filtering combining similarity and structured data  
Synchronous processing with background recovery pipelines  
Analytics layer for tracking search and usage patterns
